# Pact Broker

This recipe is to run the pact broker.

For more information : https://hub.docker.com/r/dius/pact-broker

## Quick start

```
docker-compose up
```

Pact broker : http://localhost:8080

Adminer : http://localhost:8081

Database information

| Item     |  Value   |
| -------- | :------: |
| Host     |    db    |
| Name     | postgres |
| Username | postgres |
| Password | example  |
